  • Obtaining the Source Code

The source code for the compressor, and the data analysis code, are available on GitHub here. The compressor requires a C++ compiler; it has been tested with GCC 6.3 and Clang 4.0, but should compile correctly on GCC 5. The analysis code requires Wolfram Mathematica, and has at present only been tested on Mathematica 11.
